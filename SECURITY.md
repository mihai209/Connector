# Security Policy

## Supported Platforms

Connector-GO can run on the following operating systems:

| Operating System | Status |
|------------------|--------|
| Windows (WSL) | Supported |
| Debian / Ubuntu | Supported |
| Arch Linux | Experimental |

---

## Supported Versions

Only the most recent versions receive security updates and support.  
Older releases are considered **end-of-life** and should not be used in production.

| Version | Status |
|--------|--------|
| Connector v1.4.1 (latest) | Supported |
| Connector v1.4-3-dev | Development |
| Connector v1.4 beta | Unsupported |
| Connector v1.4-patch | Unsupported |
| Connector v1.1-patch | Unsupported |
| Connector v1.1 | Unsupported |
| Connector v1.1 (Minecraft GUI Editor) | Unsupported |
| Connector v1.0 RELEASE | Unsupported |
| Connector v1.0 | Unsupported |
| Connector v1.0.4beta | Unsupported |
| Connector v1.0.1beta | Unsupported |
| Connector v1.0 [STABLE] | Unsupported |
| Connector beta v1.0 | Unsupported |

---

## Reporting a Vulnerability

If you discover a security vulnerability in **Connector-GO**, please report it responsibly.

You can report vulnerabilities through:
- **GitHub Issues** (recommended)
- **Discord**

When reporting a vulnerability, please include:
- a clear description of the issue  
- steps required to reproduce the problem  
- affected version(s)  
- possible impact if known

---

## Security Recommendations

- Always keep **Connector-GO updated to the latest version**.
- Older versions may contain **unpatched security vulnerabilities**.
- Servers running outdated releases are **not supported** and may be at risk.

Security issues reported for **obsolete versions** may not receive a response.
